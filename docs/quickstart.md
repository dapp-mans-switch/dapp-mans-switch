# Start Dapp Man's Switch From Scratch

1) `git clone https://github.com/dapp-mans-switch/dapp-mans-switch.git`
2) `cd dapp-mans-switch`
3) `npm install`
4) `dfx start [--clean]`
5) `dfx deploy`
6) `[npm start]` for fast local frontend debugging


## Troubleshooting

1) `rm -rf node_modules .dfx`
2) goto 3) above
